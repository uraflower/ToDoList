## 기능

### 할 일 추가

- 할 일의 제목 또는 내용을 적고 `+` 버튼을 누르면 목록에 추가할 수 있습니다.
- 할 일의 중요도를 0~3 단계의 별표로 나타낼 수 있습니다.

### 완료한 일 체크

- 목록에 있는 할 일의 체크 버튼을 눌러, 할 일을 완료하였음을 나타낼 수 있습니다.

### 할 일 삭제

- 목록에 있는 할 일의 `X` 버튼을 눌러, 할 일을 삭제할 수 있습니다.
- `CLEAR` 버튼을 누르면 목록의 모든 할 일들을 삭제할 수 있습니다.
